                🚗 AUTO MIRAL - Application Mobile NativeScript

## 📌 Description
AUTO MIRAL est une application mobile développée avec NativeScript et TypeScript permettant :

- l’inscription des utilisateurs
- la connexion sécurisée
- l’affichage d’une liste de voitures de luxe
- la consultation des détails des voitures
- la déconnexion

L’application communique avec une API Node.js/Express et une base de données MySQL.

---

- Node.js
